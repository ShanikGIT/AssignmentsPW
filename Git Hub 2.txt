1. How to check if git is available on your system?
Ans: git --version
2. How to initialize the new Git repository?
Ans: git init project
 cd project
3. How to tell git about your name and email?
Ans: git config --global user.name "<your name>"
git config --global user.email "<your email>"
4. How to add the file to the staging area?
Ans: git add <file>
5. How to remove the file from the staging area?
Ans: git reset HEAD -- <file>
6. How to make the commit?
Ans: git commit -m "Commit message".
7. How to send your changes to h remote repository?
Ans: git push
8. What is the difference between clone and pull?
Ans. The clone command in git is used when you want to download an existing git repository to your local computer.
 git pull When you want to make changes or updates done by another developer/team member on a git
